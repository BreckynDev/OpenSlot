# OpenSlot

## Tech Stack

- Node.js, Express, TypeScript
- PostgreSQL (Neon)
- React (in progress)

## Features

- Appointment booking with client upsert logic
- Four-table relational schema with junction table
- Status tracking (pending/confirmed/cancelled)

## Setup

1. Clone the repo
2. Run `npm install`
3. Create a `.env` file using `.env.example` as a reference
4. Run `npm run dev`
